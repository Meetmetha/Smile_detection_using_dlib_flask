Smile Detection Demonstration Using Dlib Library Modules

This application detects smiley face of person posed within the window, in real-time. It also detects whether the person in the window wear the mask or not. The application also stores the image with smiley face which we can see by just clicking the below provided link. This is developed and implemented using OpenCV, a real-time Open Source Computer Vision Library.
